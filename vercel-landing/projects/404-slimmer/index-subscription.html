404 Slimmer

Lightweight 404 error manager – detect, log, and redirect broken links to reduce server load and improve SEO.

Free & Pro WordPress Plugin

Features

  • ✓ 404 Logging — Automatically logs all 404 errors with timestamps, requested URLs, referrer, and user agent.
  • ✓ Smart Redirects — Set up permanent (301) or temporary (302) redirects from the log entries with a single click.
  • ✓ Log Viewer — Clean, paginated admin panel to browse, search, and filter 404 entries.
  • ✓ Email Notifications — Get notified when 404 errors are detected Pro
  • ✓ CSV Export — Export logs for external analysis Pro
  • ✓ SEO Friendly — Helps reduce crawl errors and improve user experience by fixing broken links.
